# Story Driven Vision - IR Template

This is a template repository for Beacon IR's Story Driven Vision template structure.

## Structure

This template defines the layout and component arrangement for IR sites using the Story Driven Vision structure.

## Usage

This template is used by Beacon IR's platform to generate client IR sites. When a client selects this template during onboarding, this repository is used as the base for their site.

## Components

The template uses Beacon IR's component composition system:
- Core IR Components (Hero, KPIs, Filings, Earnings, etc.)
- Modular themes (applied separately)
- CMS integration (Payload CMS)

## Customization

Clients can customize:
- Theme (colors, fonts, effects)
- Content (via CMS)
- Branding (logo, colors)

The template structure itself is fixed to ensure consistency and maintainability.
